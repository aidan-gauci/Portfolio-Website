**How to Run:**

1. Ensure you have a modern browser installed.
2. Clone the repository:
```git clone https://github.com/aidan-gauci/Portfolio-Website.git```
3. Navigate to repository directory: 
```cd Portfolio-Website```
4. Open **index.html** in your browser.

**Limitations:**

* Website supports any dimensions of any device width above 320px. Anything below this will break the website.
